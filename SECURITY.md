## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security issue, please report it to us responsibly.

**Do not report security vulnerabilities through public GitHub issues.**

### How to Report
- **Email**: 3890320020@qq.com
- **Private Report**: https://github.com/crazying-dev/fairy-forum/security/advisories/new

Please include the following details:
- Type of vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

### Response Process
- We will respond to your report within **7 days**.
- We will provide updates on the fix progress regularly.
- If the vulnerability is confirmed, we will release a patch as soon as possible.
- If declined, we will explain the reason.

Thank you for helping keep our project safe.
