# 贡献指南（Contributing）
感谢你对本项目的关注！我们欢迎各类贡献：**代码优化、文档完善、bug 修复、新功能提案、测试用例补充**等。
参与本项目即表示你同意遵守我们的[行为准则](./CODE_OF_CONDUCT.md)。

## 贡献类型
-  **Bug 修复**：提交 issue 反馈问题，或直接提交 PR 修复
-  **新功能**：先开 issue 讨论方案，确认后再开发
-  **文档**：补充使用说明、注释、教程
-  **测试**：补充单元测试、集成测试
-  **样式**：代码格式化、UI 细节优化（不改动核心逻辑）

## 行为准则
- 尊重他人，友善沟通，禁止人身攻击、歧视言论
- 避免无意义的刷屏、灌水
- 所有交流与贡献需符合开源社区基本礼仪
（如需详细规则，可单独创建 CODE_OF_CONDUCT.md 并链接）

## 如何提交 Issue
提交前请**先搜索历史 issue**，避免重复。
